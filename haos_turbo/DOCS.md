# Haos Turbo

## 配置说明


### 使用后台服务:

- 选择后台服务的内核类型

### 配置版本:

- 使用配置文件的版本. 目前v1的兼容性较好。

### 升速域名:

- (选项)升速的域名列表。

### 直连域名：

- (选项)直连的域名列表。

### 升速IP(CIDR)：

- (选项)升速的IP地址(支持CIDR)。

### 直连IP(CIDR)：

- (选项)直连的IP地址(支持CIDR)。

### 关闭部分日志：

- 关闭部分日志（注意：调试时才开，否则日志会占用磁盘空间）

### 清除用户数据：

- 每次启动自动清除所有的用户数据（注意：卸载插件，用户数据不会删除）

### DNS解析模式：

-  这里设置DNS解析模式。
