# 冬瓜版加载项: 



### 关于

应不少朋友的需求，特地出一个经过整理和优化的适合国内的加载项合集。

合集本着实用为主，并对合集在国内使用上，介绍等都进行了汉化或者中文化介绍为主。



### 安装方法

可以用以下方式对仓库进行添加：

在家庭助手中进行设置，配置--加载项--加载项商店--仓库--“管理加载项Repo“页面中添加仓库的URL：

https://gitcode.com/waxgourd/addons

添加后需要刷新网页，就可以找到冬瓜加载项，点击想要安装的加载项，进行安装即可。
